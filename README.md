# YouQueue

Connect and add videos to a shared YouTube playlist to be played in desired order on another device.

1. QR Code scan to go to app.

2. Taken to 'Playlist View' (Main Application view).

3. Click 'Add to Queue' button - Option to search for video to play comes up.

4. After desired video shown, 'Add video' button available.

5. Video is added to list.
